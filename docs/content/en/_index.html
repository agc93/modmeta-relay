{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} ModMeta Relay is an extensible server platform for handling mod metadata in clients using the ModMeta format. In particular, this API works as a Metaserver for Vortex, providing more options for mod metadata for non-Nexus mods. **This project is unofficial!** We have no association with Nexus Mods and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Fully Vortex compatible" %}} We support the same basic API as modmeta-db, so we're fully compatible with Vortex. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cogs" title="Extensible" %}} ModMeta Relay includes built-in plugin support for extending the server with any mod source. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/modmeta-relay" %}} ModMeta Relay is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
